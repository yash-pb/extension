YouTube Auto Pause & Resume Extension

Automatically pause and resume YouTube videos when switching tabs or windows with this handy Chrome extension.

Features

  • Pause YouTube videos automatically when switching to another tab.
  • Resume playback when you return to the YouTube tab.

Instructions

  1. Install the extension by clicking the "Add to Chrome" button above.
  2. Play a YouTube video in one tab.
  3. Switch to another tab, and the video will pause automatically.
  4. Return to the YouTube tab to resume playback.

Instructions for disable this extension temporary

  1. Right click on extension or there icon.
  2. Then go to "This can read & change site data".
  3. Select "When you click the extension".
  4. So, if will turn off. If you want to start it click on extension icon.
